# @ekairos/toolbar

Lightweight feedback toolbar for selecting UI elements and exporting structured annotations.

## Install

```bash
pnpm add @ekairos/toolbar
```

## Usage

```tsx
import { EkairosToolbar } from "@ekairos/toolbar";

export function App() {
  return (
    <>
      <YourApp />
      <EkairosToolbar />
    </>
  );
}
```

## Included

- single and multi selection
- drag area selection
- feedback dialog
- selector extraction
- markdown export

## Shortcuts

- `Cmd/Ctrl + Shift + F`
- `Esc`
- `C`
- `S`
- `X`
- `H`
